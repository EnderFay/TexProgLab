Сначала надо создать папку build

cd .../TexProgLab    # ... - это путь, где у тебя лежит папка TexProgLab #
mkdir build
cd build
cmake .. -G "MSYS Makefiles"
cmake --build .


Если ошибки выдает, попробуй это, мне помогло ->
Открываешь Пуск -> MSYS2 UCRT64 -> пишешь в консоли:
cd .../TexProgLab
pacman -Syu  
pacman -S mingw-w64-ucrt-x86_64-cmake   # скорее всего придется 2 раза прописать  pacman -Syu , если не сработает эта строка
cmake --version
g++ --version

Если всё ок, то пишешь это:
cd .../TexProgLab
mkdir build
cd build
cmake .. -G "MSYS Makefiles"
cmake --build .



в этом же терминале прописываешь:
./server.exe

cd .../TexProgLab/build
